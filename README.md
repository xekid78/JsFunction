# JsFunction
関数

## 処理
sum_gokei()関数を作り、その中で計算を行う。

## コード
```
(function() {
    'use strict'

    function sum_gokei() {
        console.log(5 * 8);
    }

    sum_gokei();

})();
```

## 出力結果  
```
40
```
